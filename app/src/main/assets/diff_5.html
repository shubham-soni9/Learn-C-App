Serial No. Basis of distinction ARRAY POINTERS
1. Declaration //In C++
type var_name[size];
//In Java.
type var-name[ ];
var_name = new type[size]; 		//In C++
type * var_name;
2. Working Stores the value of the variable of homogeneous datatype. Store the address of the another variable of same datatype as the pointer variable's datatype.
3. Generation An array of pointers can be generated. A pointer to an array can be generated.
4. Java Support Support the concept of array. Does not support pointers.
5. Storage A normal array stores values of variable and pointer array stores the address of variables. Pointers are specially designed to store the address of variables.
6. Capacity An array can store the number of elements, mentioned in the size of array variable. A pointer variable can store the address of only one variable at a time.
