Compile and Run C Program

To compile and run a C language program, you need a C compiler. To setup a C language compiler in your Computer/laptop, there are two ways:

  1. Download IDE like Turbo C or Microsoft Visual C++
  2. Or, you can use any text editor to edit the program files and download the C compiler separately
Using an IDE - Turbo C

To compile and run a C language program, you need a C compiler. To setup a C language compiler in your Computer/laptop, there are two ways:

  1. Open turbo C IDE(Integrated Development Environment), click on File and then click on New
  2. Write the program in editor
  3. Click on compile or press Alt+f9 to compile the code
  4. Click on Run or press Ctrl+f9 to run the code
  5. If there is no error. You will see the output in black screen
